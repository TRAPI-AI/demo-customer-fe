Based on the Duffel API documentation, here are the required parameters for the payload object and the response data object:

**Payload Object:**

```json
{
  "data": {
    "slices": [
      {
        "origin": "string", // Required. The IATA code of the origin airport.
        "destination": "string", // Required. The IATA code of the destination airport.
        "departure_date": "string" // Required. The date of departure in the format "YYYY-MM-DD".
      }
    ],
    "passengers": [
      {
        "type": "string" // Required. The type of passenger. Possible values: "adult", "child", "infant", "young_adult", "senior".
      }
    ]
  }
}
```

**Response Data Object:**

```json
{
  "data": {
    "slices": [
      {
        "origin_type": "string", // The type of the origin. Possible values: "airport", "city".
        "origin": {
          "type": "string", // The type of the origin. Possible values: "airport", "city".
          "id": "string", // Duffel's unique identifier for the origin.
          "iata_code": "string", // The IATA code of the origin.
          "iata_country_code": "string", // The IATA code of the country of the origin.
          "iata_city_code": "string", // The IATA code of the city of the origin.
          "name": "string", // The name of the origin.
          "city_name": "string", // The name of the city of the origin.
          "time_zone": "string", // The time zone of the origin.
          "longitude": "number", // The longitude of the origin.
          "latitude": "number" // The latitude of the origin.
        },
        "destination_type": "string", // The type of the destination. Possible values: "airport", "city".
        "destination": {
          "type": "string", // The type of the destination. Possible values: "airport", "city".
          "id": "string", // Duffel's unique identifier for the destination.
          "iata_code": "string", // The IATA code of the destination.
          "iata_country_code": "string", // The IATA code of the country of the destination.
          "iata_city_code": "string", // The IATA code of the city of the destination.
          "name": "string", // The name of the destination.
          "city_name": "string", // The name of the city of the destination.
          "time_zone": "string", // The time zone of the destination.
          "longitude": "number", // The longitude of the destination.
          "latitude": "number" // The latitude of the destination.
        },
        "departure_date": "string" // The date of departure in the format "YYYY-MM-DD".
      }
    ],
    "id": "string", // Duffel's unique identifier for the offer request.
    "created_at": "string", // The ISO 8601 datetime at which the offer request was created.
    "live_mode": "boolean" // Whether the offer request was created in live mode.
  }
}
```

Please note that the actual response may contain more data fields depending on the specific request and the data provided by the airlines.
